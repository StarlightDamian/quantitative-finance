## 交易单 transaction_order_df



## 买入

|  参数名称  |  参数描述  |                说明                 | 备注 |
| :--------: | :--------: | :---------------------------------: | :--: |
|    date    |  交易日期  |          格式：YYYY-MM-DD           |      |
|    time    |  交易时间  |       格式：YYYYMMDDHHMMSSsss       |      |
|    code    |  证券代码  | 格式：sh.600000。sh：上海，sz：深圳 |      |
|    open    | 今开盘价格 |  精度：小数点后4位；单位：人民币元  |      |
|   volume   |  成交数量  |              单位：股               |      |
|   amount   |  成交金额  |  精度：小数点后4位；单位：人民币元  |      |
| unit_price |  成交单价  |         成交金额 / 成交数量         |      |
